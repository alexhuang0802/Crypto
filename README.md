# 🚀 Crypto — MACD 背離 + 資費套利 掃描工具

這是一個針對加密貨幣市場設計的 **趨勢預判與資費套利分析工具**。  
核心邏輯是透過 **MACD 背離偵測** 來篩選出「即將啟動大走勢」的幣種，  
同時輔助觀察 **持倉量** 與 **資費變化**，  
協助交易者在多空轉換階段提前佈局。

---

## 🧩 功能特色
- 🔍 **MACD 背離偵測**
  - 偵測多方與空方背離訊號
  - 抓出「價格未破底但 MACD 已轉強」的潛在反轉幣
- 💹 **持倉量 + 資費監控**
  - 即時顯示各大合約資費變化（特別支援 BingX）
  - 幫助使用者判斷市場偏多或偏空
  - 若想「領取資費」，可根據資費排行選擇方向套利
- 📊 **多時間框架支援**
  - 1小時 / 4小時 / 日線
  - 適用短線與波段策略
- ⚙️ **自動化掃描**
  - 可整合 Telegram Bot 推播
  - 自動更新最新異動幣種清單
